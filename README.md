bTabs
=====

jQuery Tabs Plugin on Diet.

> Still work in progress :)

## Demo
View initial [DEMO](http://object505.github.io/bTabs/)

## Getting Started
Getting started is super simple: include jQuery, include btabs, then:
```javascript
jQuery('#btabs').bTabs(); 
```
